# Mad lib app

* React app for mad libs game

## Run in production
```bash
npm start
```

## Run while developing
* In one tab
```bash
npm run build-dev
```
* In another tab
```bash
npm start
```

## Test (Front-end)
* In another tab, while the server is running.
```bash
npm run test-fe
```
* This creates `reports/screenshots`
